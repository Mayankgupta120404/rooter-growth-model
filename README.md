# Rooter Growth Strategy Model

## 📌 Overview
This project presents a structured growth analysis for Rooter Shop and a tournament engagement framework for BGMI users.

It includes:
- Profit impact model (Frequency vs AOV)
- Strategic recommendation
- Sensitivity analysis
- Tournament retention framework
- Behavioral psychology insights

---

## 📊 Q1: Growth Lever Analysis

### Base Metrics
- MAU: 1M
- Orders/User: 1.6
- AOV: ₹1500
- Gross Margin: 5%
- Net Profit: 3%

### Levers Evaluated
1. Increase Purchase Frequency
2. Increase AOV (Upsell + Bundling)

### Key Insight
- Frequency → Higher absolute profit
- AOV → Faster execution, lower cost, better short-term fit

### Recommendation
👉 AOV Increase (Product-led growth)

---

## 📉 Stress Test
Critical variable: Upsell Conversion Rate  
Break-even: ~5–6%

---

## 🎮 Q2: Tournament Retention Problem

### Core Issue
- 100 winners out of 100,000 users
- 99.9% users feel unrewarded

### Psychological Drivers
- Learned Helplessness
- Expectancy Collapse
- Lack of Progress Visibility

---

## 🧠 Solution: 5-Tier Reward System

| Tier | Users | Strategy |
|------|------|---------|
| T1 | Top 100 | Cash + Status |
| T2 | 101–1000 | Recognition |
| T3 | 1K–10K | Progress feedback |
| T4 | 10K–50K | Community |
| T5 | 50K–100K | Fun + retention |
